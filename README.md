## Usage
* First step is to install jupyter notebook. This can be done by running this command: "pip install jupyter".
* Next step is to run jupyter notebook by running this command in the terminal: "jupyter notebook".
* This will open jupyter notebook in the web browser.
* From the web browser, move into the directory where you have downloaded the repository.
* Now open the project_1.ipynb file. The jupyter notebook will open and you can run the cells to replicate the project.

## Downloading the project
* Create a folder where you want to download the project.
* Now open terminal or command prompt in that folder.
* Download the project from github using this command: "git clone git@github.com:1993jayant/music-emotion-prediction.git".
* Another option is to download the zip file of the project from project's github repository, and then extract the project files.
* You first need to install the dependencies of the project.
* In order to install the dependencies, run this command in the terminal: "pip install -r requirements.txt"
